.Section attributes and values
[cols="1m,1,2m,2"]
|===
|Attribute |Value(s) |Example Syntax |Comments

|sectid
|N/A
|+:sectid!:+
|Intrinsic; set to autogenerate by default

|idprefix
|`_`, or user defined text
|+:idprefix: -+
|Intrinsic; set to autogenerate a `_` by default

|sectanchors
|N/A
|+:sectanchors:+
|Asciidoctor only

|sectlinks
|N/A
|+:sectlinks:+
|Asciidoctor only

|sectnums
|N/A
|+:sectnums:+
|Section numbering is off by default.
Can be toggled on or off through document.

|sectnumlevels
|1-9
|+:sectnumlevels: 3+
|Controls the depth of section numbering (both at the section title and in the outline). The default value is 3.

|===
